# S²-VLA: State-Space Guided Vision-Language-Action Models for Long-Horizon Manipulation

This repository contains the official project page code for **S²-VLA: State-Space Guided Vision-Language-Action Models for Long-Horizon Manipulation**. The landing page introduces our framework, tracking core metrics across real-world ALOHA platforms alongside both LIBERO and SimplerEnv simulation benchmarks.

The project page is built by modifying the [Academic Project Page Template](https://github.com/eliahuhorwitz/Academic-project-page-template).
